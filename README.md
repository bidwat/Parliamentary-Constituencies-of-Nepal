## Dataset File

- `nepal_election_constituencies_area.json`

## JSON Structure

The JSON is an array of district objects.

Each district object has:

- `district` (string)
- `constituencies` (array)

Each constituency object has:

- `name` (string)
- `population` (integer electorate)
- `municipalities` (array)
  - each municipality has `name`
  - optional `wards` array when only specific wards are included

### Notes

- If `municipalities` is empty, that constituency covers the entire district.
- The same municipality can appear in multiple constituencies with different ward lists.

## Example

```json
{
  "district": "Ilam",
  "constituencies": [
    {
      "name": "Ilam 1",
      "population": 109535,
      "municipalities": [
        { "name": "Sandakpur Rural Municipality" },
        { "name": "Ilam Municipality", "wards": [10, 13, 14] }
      ]
    }
  ]
}
```
